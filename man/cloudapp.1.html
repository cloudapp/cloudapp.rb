NAME

cloudapp - All the pleasures of CloudApp now at your terminal

SYNOPSIS

cloudapp [--format=format] command [args]

cloudapp list [--count=count] [--filter=all,active,trash]
cloudapp details href [href...]
cloudapp download [--output=file] url

cloudapp bookmark [--private] [--public] url [url...]
cloudapp upload [--private] [--public] file [file...]
cloudapp update [--name=name] [--bookmark=bookmark_url] [--private] [--public] href [href...]

cloudapp trash href [href...]
cloudapp recover href [href...]
cloudapp delete href [href...]

DESCRIPTION

cloudapp is a simple, Unix-friendly tool for CloudApp meant to augment the Mac and Web apps. Sometimes it's more convenient to share without leaving the terminal and other times it's the only tool available.

GLOBAL OPTIONS

-f format, --format=format
Output using either pretty or csv format. (Default: pretty when output to a terminal and csv when output is piped to another program.)

COMMANDS

bookmark [--private] [--public] url [url...]

Create a new bookmark to each url and print the share links.

delete href [href...]

Permanently delete each drop at href.

details href [href...]

View the details of each drop at href.

download [--output=file] url

Download the drop's content at url to file. The default is to download to the current directory.

list [--count=count] [--filter=all,active,trash]

List the newest count of filter drops. The default is to list 20 active drops.

recover href [href...]

Recover each drop at href from the trash.

trash href [href...]

Send each drop at href to the trash.

update[--name=name] [--bookmark=bookmark_url] [--private] [--public] href [href...]

Update the name, privacy, or bookmark link of each drop at href.

upload [--private] [--public] file [file...]

Upload each file and print the share links.

COMMAND OPTIONS

--filter=all,active,trash

Show filter drops. (Default: active)

-n count, --count=count

Display count drops. (Default: 20)

-o file, --output=file

Download file as file. (Default: current directory using the remote file's name)

--private

Use a private (long) link when creating a new drop.

--public

Use a public (short) link when creating a new drop.

REQUIREMENTS

cloudapp requires Ruby 1.9.2 or greater.

$ cloudapp list

Chapter 1.md            http://api.getcloudapp.com/...  http://cl.ly/3U3d...  7
CloudApp                http://api.getcloudapp.com/...  http://cl.ly/2O2J...  3
  ...

List the 5 newest drops:

$ cloudapp list --count=5
Name                    Href                            Share                 Views
favicon.ico             http://api.getcloudapp.com/...  http://cl.ly/0P28...  3
The Hitchhiker's Guide  http://api.getcloudapp.com/...  http://cl.ly/3I28...  4
CloudApp 1.5 Released   http://api.getcloudapp.com/...  http://cl.ly/0s1o...  4
Cover.jpeg              http://api.getcloudapp.com/...  http://cl.ly/1x20...  1
Chapter 1.md            http://api.getcloudapp.com/...  http://cl.ly/3U3d...  7

List newest, deleted drops:

$ cloudapp list --filter=trash
Name       Href                            Share                 Views
Delete Me  http://api.getcloudapp.com/...  http://cl.ly/1H2C...  0
  ...

List newest 5 drops in CSV format:

$ cloudapp --format=csv list --count=5
Name,Href,Share,Views
favicon.ico,http://api.getcloudapp.com/...,http://cl.ly/0P28...,3
The Hitchhiker's Guide to the Galaxy,http://api.getcloudapp.com/...,http://cl.ly/3I28...,4
2011-05-03-cloudapp-1.5-released.md,http://api.getcloudapp.com/...,http://cl.ly/0s1o...,4
Cover.jpeg,http://api.getcloudapp.com/...,http://cl.ly/1x20...,1
Chapter 1.md,http://api.getcloudapp.com/...,http://cl.ly/3U3d...,7

View the details of a drop give its href:

$ cloudapp http://api.getcloudapp.com/drops/12345
Name:     Cover.jpg
Created:  2 days ago
Privacy:  private
Share:    http://cl.ly/1x20...
Emebed:   http://cl.ly/1x20...
Download: http://cl.ly/1x20...

View the details of a drop in CSV format:

$ cloudapp --format=csv http://api.getcloudapp.com/drops/12345
Name,Created,Privacy,Share,Embed,Download
Cover.jpg,2012-01-12T20:44:58Z,private,http://cl.ly/1x20...,http://cl.ly/1x20...,http://cl.ly/1x20...

Share a new bookmark to http://douglasadams.com:

$ cloudapp bookmark http://douglasadams.com
Bookmarking http://douglasadams.com... http://cl.ly/1y0j403g3D0c0X1G0R3m

Share the file screenshot.png:

$ cloudapp upload screenshot.png
Uploading screenshot.png... http://cl.ly/040u2o3X1w0z1z3n2T04

Share a new bookmark or file and copy the URL to the clipboard (OS X):

$ cloudapp bookmark http://douglasadams.com | pbcopy
$ cloudapp upload screenshot.png | pbcopy

Share a new bookmark and output only the URL:

$ cloudapp --format=csv bookmark http://douglasadams.com
http://cl.ly/1y0j403g3D0c0X1G0R3m

Download a drop to the current directory:

$ cloudapp download http://cl.ly/040u2o3X1w0z1z3n2T04
Downloading screenshot.png... done

Download a drop to another path:

$ cloudapp download --output=/Users/Larry/image.png http://cl.ly/040u2o3X1w0z1z3n2T04
Downloading screenshot.png to /Users/Larry... done
